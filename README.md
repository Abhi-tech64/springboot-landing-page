# springboot-landing-page

This project is a simple full stack application built using:

- HTML
- CSS
- Java Spring Boot

## Features
- Landing page UI
- Contact form
- Backend form handling
- Console logging of submitted data

## How It Works
1. User fills the contact form.
2. Form sends POST request to Spring Boot backend.
3. Backend controller receives data.
4. Data is printed in the console.

## Technologies Used
- HTML
- CSS
- Java
- Spring Boot

Tech Stack: HTML, CSS, Java, Spring Boot, MySQL, JPA
